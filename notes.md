### Text Formating
- Bold: \*\* \*\* or \_\_ \_\_
- Italic: \* \* or \_ \_
- Strikethrough: \~\~ \~\~
- Inline Code: \` \`
- Code Blocks: \`\`\` \`\`\`

### SSH
`ssh -i [key pair file] ubuntu@[ip address]`

If you get a warning that your key pair file permissions are too open run the `chmod` command:

```chmod 600 [key pair file]```
